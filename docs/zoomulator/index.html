Features

動物園を自由に育てよう

30種以上の動物

ライオン、ゾウ、チーター、フラミンゴ…希少な神獣も登場。ガチャで仲間を集めよう。

5種類の区画

サバンナ・森林・北極・湿地・荒れ地。それぞれに合った動物と設備でカスタマイズ。

コンボボーナス

特定の動物・設備の組み合わせで隠れたコンボを発動。来場者数が一気に跳ね上がる！

四季とイベント

春夏秋冬で来場者数や動物の魅力が変化。季節限定イベントも発生。

20種以上の来場者

家族連れ・カメラマン・研究者・忍者…動物の組み合わせで珍しい来場者が現れる。

30段階の動物園ランク

累計来場者数に応じてランクアップ。ランク賞金と新しい動物・設備が解放される。
