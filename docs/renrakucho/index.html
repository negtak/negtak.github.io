Features

ふたりの暮らしを、ひとつに

📝

れんらく

「洗剤が切れた」「牛乳を買って」。買い物や連絡をリストで共有。チェックして完了、まとめて消せる。

📌

きめごと

「ゴミ出しは交代」などのルールや取り決めを、消えないノートとして永続的に記録。

💰

おかね

支出を金額・内容・支払者で記録。メンバー別＆全体の合計を自動で集計。

🔗

かんたん連携

片方が作成して6桁の招待コードを共有するだけ。相手はコードを入れて参加。

⚡

リアルタイム同期

追加・チェックした内容は相手の画面にもすぐ反映。離れていても同じノートを開ける。

🙂

はじめやすい

名前を入れるだけですぐスタート。むずかしい登録は不要です。
