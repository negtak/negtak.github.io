Features

恐竜園を自由に育てよう

🦕

多彩な恐竜たち

ティラノサウルス、トリケラトプス、ブラキオサウルス…伝説の古代生物も登場。ガチャで仲間を集めよう。

🌋

時代別エリア

三畳紀・ジュラ紀・白亜紀など、それぞれの時代に合ったエリアを設計しよう。

✨

コンボボーナス

特定の恐竜・設備の組み合わせで隠れたコンボを発動。来場者数が跳ね上がる！

🌿

太古のイベント

季節ごとに特別な恐竜が登場。期間限定の発掘イベントを楽しもう。

👨‍👩‍👧

個性豊かな来場者

家族連れ・古生物学者・冒険家…恐竜の組み合わせで珍しい来場者が現れる。

🏆

恐竜園ランク

累計来場者数に応じてランクアップ。新しい恐竜や設備が次々と解放される。
